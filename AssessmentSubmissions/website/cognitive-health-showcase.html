About the Project

Cognitive decline often begins long before clinical symptoms emerge — the goal is to detect it earlier, more unintrusively, and with greater precision than existing methods allow.

This project proposes and implements an intelligent cognitive health monitoring system that passively observes behavioral signals — typing rhythm, navigation patterns, micro-interaction timing, and task-switching behavior — to build a longitudinal model of an individual's cognitive performance.

Unlike clinical assessments that require scheduled appointments, this system operates continuously in the background, providing an ambient, non-intrusive health intelligence layer. When deviations from an individual's personal baseline are detected, the system delivers gentle, contextual ambient feedback — subtle audio cues, lighting variations, or interface nudges — rather than alarming notifications.

The system was evaluated on public datasets: CMU Dataset and Balabit Mouse Dynamics Dataset, and demonstrates strong intra-individual consistency and meaningful inter-individual discriminability. Models achieve classification performance while remaining computationally lightweight enough for edge deployment.
